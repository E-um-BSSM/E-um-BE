## 인증 시스템

- **BSM OAuth 연동**
    - 기존 BSM 계정으로 간편 로그인 (학생 + 졸업생)
- **자체 회원가입**
    - 예비 신입생 대상, 이메일/전화번호 인증 포함
    - 어드민용
- **멘토 및 멘티**
    - 멘토: 클래스 개설자 및 권한을 부여 받은 사람
    - 멘티: 클래스 참여자
- **역할별 권한 관리**
    - 멘토: 관리, 후기 열람,
    - 멘티: 클래스 참여, 후기 작성
    - 어드민: 전체 관리 권한
    - 예비 신입생: 멘티와 동등한 권한 + 전용 뱃지 제공
- **비밀번호**
    - 비밀번호 재설정 / 찾기
- **로그인 유지**
    - JWT 사용

## 멘토링 클래스 관리

- **모집글 생성 + 수정,삭제,조회**
    - 기본적인 폼을 제공해주고 그 상태로 작성 후 올림
    - 모집글 올린 후 자신 포함 2명 이상의 신청자가 모이면 클래스 개설이 가능
    - 클래스 개설 후 모집글을 닫히고 클래스가 개설 됨 그리고 클래스 초대 코드가 클래스에 지정됨 즉, 클래스 개설 이후 참여하고 싶은 학생은 따로 코드로 참여.
- **클래스 생성**
    - 고유 클래스 ID 자동 생성
    - 주제, 설명, 일정, 난이도 태그 설정
- **클래스 업데이트**
    - 과제 등록/수정
    - 일정 변경 시 알림 발송
    - 참여 인원 조정(강퇴/재등록 가능)
- **클래스 삭제**
    - 완전 삭제 or 보관
- **클래스 조회**
    - 범위 → 삭제, 모든 클래스는 초대코드로 참가 가느으
    - 검색
        - 태그별 검색
        - 난이도별 검색
- **클래스 초대**
    - 클래스 초대코드로 참가하면 대기 리스트로 감
    - 멘토가 대기 리스트에서 참가자 고를 수 있음
- **클래스 참여**
    - 클래스 만든 사람은 멘토, 나머지 참여자는 모두 멘티로 참가
    - 멘토가 대기 리스트에 있는 사람들의 클래스 참여를 혀용/거부 가능
- **클래스 멤버 관리**
    - 멘토가 클래스 멘티 관리할 수 있음
    - 멘토가 멘티를 멘토로 승격할 수 있음
- **클래스 일정 관리**
    - crud
    - 각 유저마다 일정 가능 여부 확인
    - 멘토끼리 일정이 겹치면 그 멘토들에게 알람 송신
    - 일정을 생성하면 schedules에 내용을 작성하고 schedules_user에 그 클래스 맴버들 모두 생성함
- **클래스 과제 관리**
    - crud
    - 제출
        - 마감기간 지킴 = sumbitted
        - 마감기간 넘음 = late
    - 과제 확인 완료 = graded
        - 멘토가 과제 점수 채점
        - 멘토가 피드백 달기 가능
- **클래스 태그**
    - crud
- **클래스 공지 관리**
    - crud

## 커뮤니티 게시판

- **공부 자료 공유 게시판**
    - 둘다 임시 저장 기능있음
    - 채택 시스템을 제외하곤 지식인과 비슷함 api는 아마
- **뱃지 시스템**
    - 질문 많은 멘티, 인기 멘토, 성실 멘티, 예비 신입생 전용 뱃지 등
- **채택 기능**
    - 게시자는 마음에 드는 답변 채택 가능 (중복 가능)
    - 채택되면 글 작성자가 댓글 작성자에게 내공을 줌. (게시글 작성 때 걸 내공 설정)
- **댓글**
    - crud
    - 채택되면 알람 송신
- **내공 시스템**
    - 답변 채택 시 내공 획득
    - 계정 당 기본 내공: 1000
    - 내공의 양에 따른 칭호
        - **0** → 코린이
        - **500** → starter
    - 재화와 별
- **꾸미기 기능**
    - 글자 색상 변경, 이펙트 적용
- **관리**
    - crud

## 후기 시스템

- **멘토 ←→ 멘티 후기**
    - 클래스 수료 후만 작성 가능
    - 평점 대신 태그 기반
    - 태그는 하나만 선택 가능
    - (예: 열정적, 성실함, 창의적 등)
- **태그 통계 시각화**
    - 개별 수치 비공개, 비율 그래프 제공 → 대략적인 성향 파악 가능
- **후기 열람**
    - 전체 공개
    - 멘토: 받은 후기 집계하여 열람
    - 멘티: 집계 + 클래스별 후기 열람

## 재화 시스템

- **획득**
    - 멘토링 참여, 질문/답변, 일일 출석 보상을 통해 획득
- **소비**
    - 프로필 꾸미기 아이템 구매
    - 커뮤니티 꾸미기 요소 적용
- **후원**
    - 멘토-멘티 간 송금 가능 → transactions에 기록 남음
- **일일 보상**
    - 매일 재화 +1 자동 지급
